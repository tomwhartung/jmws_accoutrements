Joomla back end:
----------------
Procedure "tested" installing new module (on 8/28/2015).

Top Menu:
	Extensions -> Extension Manager

Side Menu:
	Discover

Filter:
	Site (dropdown)
	Module (dropdown)

Near top of page:
	Discover (button)

Should see new template or module in the list, else you are doing something wrong.

Select new template or module to install

Near top of page:
	Install (button)
